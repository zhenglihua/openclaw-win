## 一、功能清单

## 三、技术架构

### 3.1 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | .NET 8.0 + WPF |
| UI库 | WPF-UI 3.0.5 (Fluent Design) |
| MVVM | CommunityToolkit.Mvvm 8.2.2 |
| 服务控制 | System.ServiceProcess.ServiceController 8.0.0 |
| 日志 | Serilog |
| 部署 | 自包含单文件 (win-x64) |

### 3.2 项目结构

```
OpenClawManager/
├── Views/
│   ├── DashboardPage.xaml(.cs)    # 仪表盘页面
│   ├── InstallPage.xaml(.cs)       # 安装向导页面
│   ├── StatusPage.xaml(.cs)         # 状态监控页面
│   └── SettingsPage.xaml(.cs)       # 设置页面
├── ViewModels/
│   ├── DashboardViewModel.cs        # 仪表盘逻辑
│   ├── InstallViewModel.cs          # 安装向导逻辑
│   ├── StatusViewModel.cs           # 状态监控逻辑
│   └── SettingsViewModel.cs        # 设置逻辑
├── Services/
│   ├── ProcessService.cs            # 进程管理
│   ├── DiagnosticService.cs         # 诊断服务
│   └── NssmServiceInstaller.cs      # Windows服务安装
├── Models/
│   ├── AppConfig.cs                 # 配置模型
│   ├── DiagnosticResult.cs          # 诊断结果
│   └── OpenClawStatus.cs            # 状态模型
└── MainWindow.xaml(.cs)             # 主窗口
```
